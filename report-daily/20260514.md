## Bài học hôm nay

**Chủ đề**: Cài đặt python và học các kiến thức cơ bản về toán tử, kiểu

**Ghi chép tư duy**:
- **venv là hộp cô lập** — mỗi project 1 hộp, package không trộn lẫn, không bao giờ cài global.
-> giống npm của node js, cô lập môi trường thư viện cho từng dự án -> tránh conflict thư viện
- if __name__ == "__main__" : cách Python phân biệt "file đang được chạy" vs "file đang được dùng như thư viện"
**Kiểu và toán tử:**
- Kiểu chỉ có int và float, không nhiều kiểu như java, không bị overflow, thằng int là nguyên dynamic, float là thực
- Lũy thừa khác nhau, java dùng Math.pow, python dùng **
- Chia bình thường /, 2 thằng nguyên trả về float nếu không chia hết, java trả về nguyên nếu không ép kiểu
- Có hỗ trợ chia chỉ lấy nguyên qua toán tử //
- Có ép kiểu như java
- Có kiểm tra kiểu tương tự instance of
- Thằng boolean khác nhau ở chữ cái hoa đầu
- Truthy và Falsy giống giống javascript, 0 int hay float, string rỗng, các kiểu danh sách không có phần tử, kiểu None(tương tự null trong java) đều là rỗng
- bool là subclass của int, true bằng 1, false bằng 1 
-> là sub của int thì thực hiện phép tính bình thường, ép kiểu bình thường
- toán tử so sánh 
    - == trong java với object là so sánh reference
    - == trong python so sánh giá trị, không so so sánh reference, kể cả là phần tử trong mảng nếu cùng giá trị và thứ tự
    - python dùng toán tử is, is not (java == và !=) để so sánh reference
    - in, not in tương tự method contains của thằng list trong java, nhưng th python support kiểu toán tử
-> khá giống ngôn ngữ tự nhiên
- Java không có chained comparison kiểu 1 < x < 10, bên java sẽ tách ra dùng toán tử &&: x > 1 && x < 10 -> giống ngôn ngữ tự nhiên
- None tương tự null trong java
- Toán tử logic dùng and or not ý nghĩa same java && || !, toán tử phủ định th python dùng **not**, dùng giông java
- Đặc thù python: and/or trả về giá trị gốc, không phải True/False, th java &&/|| luôn trả về boolean
    - Gặp truthy thì đi tới khi gặp falsy hoặc cuối cùng sẽ dừng lại -> tại vị trí dừng lại sẽ trả về giá trị đó (không phải boolean)
- **Toán tử kết hợp**, khác biệt với java ở thằng /= trả ra float
    - có thêm //= chia trả ra nguyên
    - **= thằng này là dồn lũy thừa
- Không có ++ và --, thay bằng i += 1 và i -=1, tức mặc đinh bên java là i++ và i--
- Có toán tử dùng để format số, ví dụ: {price:,} -> phân tách bằng dấu phẩy, {price * tax_rate:,.0f} -> phân tách bằng dấu phẩy nhưng k lấy thập phân
- Có toán tử 3 ngôi, ví dụ {'là' if is_leap else 'không phải'}, tương tự java ("Năm " + year + " " + (isLeap ? "là" : "không phải") + " năm nhuận");
