## Bài học hôm nay

**Chủ đề**: Xử lí luồng, if else, vòng lặp, danh sách

**Ghi chép tư duy**:
- if else với python đơn giản hơn java 
    - không dấu {} -> dúng thụt lề
    - không dấu () trong expression điều kiện
    - tận dụng truthy và falsy 
- match case tương tự switch case của java
    - có _ tương tự default block của java
    - mạnh hơn java's switch
- while condition flow vẫn vậy
- Khối while else độc quyền python, khối while kết thúc tự nhiên sẽ false nhảy vào else
- Không có for dạng index như java, dùng for-each
- Có cả for-else độc quyền, tương tự while-else
- break chỉ thoát khỏi vòng lặp gần nhất bao quanh nó
- range không phải list mà là lazy sequence -> không tạo sẵn list trong bộ nhớ -> saving ram memo
- range có thể chuyển sang list bằng method list()
- List Comprehension là đặc trưng mạnh trong python
    - Mục đích **tạo collection** mới từ interable trong **1 dòng**
    - ví dụ squares = [i ** 2 for i in range(1, 6)]
    - [n for n in numbers if n % 2 == 0]
- Dict Comprehension : {key_expr: value_expr for item in iterable if condition}
    - {name: score for name, score in scores.items() if score >= 60}
    - reversed_dict = {v: k for k, v in original.items()}
- Set Comprehension: {expression for item in iterable if condition}
- Nhớ kĩ: [] list -> giá trị trùng lặp | {} dict, set -> điểm chung không trùng lặp
- Toán tử 3 ngôi có nhưng ngôn ngữ tự nhiên hơn `label = "Người lớn" if age >= 18 else "Trẻ em"`
