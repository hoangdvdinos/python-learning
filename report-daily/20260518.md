## Bài học hôm nay

**Chủ đề**: String và danh sách

**Ghi chép tư duy**:
- String python cũng immutable giống java, mọi thao tác sửa đều tạo ra object mới
- String python truy cập từng kí tự theo index giống mảng
    - Có cắt chuỗi con giống mảng, text[start:stop:step]
    - Hoa, thường, title, trip trái, trip phải, thay thế chuỗi, tìm kiếm chuỗi, kiểm tra bắt đầu, kết thúc, đếm số lần xuất hiện của chuỗi, tách chuỗi thành list, nối list thành chuỗi, 
- List trong python khác java, có thể chứa nhiều kiểu cùng lúc, ví dụ: [1, "hello", True, 3.14, None]
- CRUD với list
    - read: truy cập bằng chỉ số dương, âm
        - slicing cắt mảng: numbers[2:5], numbers[:3], numbers[::2], numbers[::-1]
    - create:
        - thêm phần tử vào cuối append
        - thêm vào index cụ thể: fruits.insert(1, "avocado")
        - nối thêm 1 list khác extend: fruits.extend(["date", "elderberry"])
    - delete:
        - xóa lần đầu xuất hiện remove: numbers.remove(1)  
        - xóa và trả về phần tử cuối .pop()
        - xóa và trả về phần tử tại vị trí chỉ định: .pop(0)
        - data.clear(): xóa sạch
    - sắp xếp
        - sắp xết tại chỗ, ảnh hưởng mảng gốc: nums.sort(), nums.sort(reverse=True), data.reverse()
        - sắp xếp trả về list mới: new_sorted = sorted(original)
- Queue và stack
    - stack(LIFO): dùng list với append/pop -> get phần tử đầu bị dịch toàn bộ mảng
    - deque(FIFO): dùng append/popleft -> get phần tử đầu không bị dịch toàn bộ mảng
- Tuple giống list nhưng immutable sau khi tạo
    - dấu phẩy là bắt buộc cho tuple 1 phần tử: single = (42,) 
    - read: truy cập giống list
    - không thể sửa
    - dùng làm key của dict được
    - return nhiều giá trị từ hàm trả về tuple
- Unpacking với *
    - first, *rest = [1, 2, 3, 4, 5]
    - print(first)    # 1
    - print(rest)     # [2, 3, 4, 5]
- Set tương tự hashset java, không thứ tự, không trùng lặp
    - từ list loại bỏ trùng lặp
    - xóa không ném lỗi nếu không tồn tại: discard
    - xóa ném KeyError nếu không tồn tại: remove
- Phép hội, giao, hiệu, sample
```Sơ đồ các phép toán:
  A = {1,2,3,4,5}    B = {4,5,6,7,8}

  A | B  = {1,2,3,4,5,6,7,8}  — tất cả
  A & B  = {4,5}               — chỉ phần chung
  A - B  = {1,2,3}             — A trừ đi phần chung
  A ^ B  = {1,2,3,6,7,8}      — không kể phần chung
```
- dict trong py tương tự hashmap, từ 3.7 giữ thứ tự chèn vào
    - ngoài khởi tạo {}, có thể khởi tạo với dict(host="localhost", port=8080, debug=True)
    - delete: 
        - del user["city"]
        - xóa và return value user.pop("age")
    - trả về danh sách key .keys() -> lưu ý trả ra không phải list, cần cast về list
    - trả về danh sách value .values()
    - trả về tuple key value product.items() -> để duyệt vòng lặp key value
    - truy cập an toàn không ném KeyError, tương tự .getOrDefault của java: product.get("discount", 0)
    - Có cả setdefault: lấy và đặt giá trị mặc định nếu key chưa tồn tại
    - Có merge dict, tức gộp 2 dict, dùng function: .update(), hoặc dùng toán toán tử |: product | {"color": "black"}
    - Có cả built-in support đếm số lần xuất hiện của phần tử, return dict
